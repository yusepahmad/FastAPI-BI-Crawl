## Getting Started

Deployment REST API to Docker

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/yusepahmad/FastAPI-BI-Crawl.git
   ```

2. Built image
   ```sh
   sudo docker build -t [name_image] .
   ```

3. Run container
   ```sh
   sudo docker run -d -p 8016:8016 nama_image
   ```

3. Access API in web
   ```sh
   http://0.0.0.0:8016/docs
   ```
